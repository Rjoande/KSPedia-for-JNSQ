# KSPedia for JNSQ

*KSPedia for JNSQ* provides in-game reference pages tailored to the JNSQ planet pack.  
It includes:
- A complete **Delta-V Map** for the rescaled JNSQ system
- **Antenna Distances** based on OhioBobâ€™s data, adapted for KSPedia  
- A **Planet Wiki** inspired by the original work of jandcando 

These pages appear directly inside the KSPedia, offering quick access during flight and mission planning.

## Installation
1. Download the latest release.  
2. Drop the file `JNSQ_KSPedia.ksp` into your `GameData` directory.  
3. Launch the game. The new KSPedia pages will appear automatically.

Tested on **KSP 1.12.x** with **JNSQ 0.10+**.
